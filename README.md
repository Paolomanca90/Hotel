## Usage 3ï¸âƒ£ <span id="usage"></span>

The app does not require login as it is for internal use only. The receptionist can enter a new reservation for a customer by adding their personal data and selecting a room. The cost will be calculated based on the choice of single or double room. Furthermore, extra services can be added later based on customer request.
The checkout page instead shows the total calculated based on the type of accommodation, the number of days and any extras.
